Rules

  1. Game Objective:
    • The user must answer 10 multiple-choice questions correctly to win $1,000,000.
  2. Difficulty & Category:
    • The user can choose the difficulty level of the game (e.g., Easy, Medium, Hard).
    • The user can also select a category (e.g., General Knowledge).
  3. Starting the Game:
    • To begin the game, the user needs to press the "Start" button.
  4. Answering Questions:
    • Each question has four possible answers (Answer A, Answer B, Answer C, and Answer D).
    • Only one of these answers is correct for each question.
  5. Progression:
    • As the user answers questions correctly, they move up the prize ladder, starting from
    • $0 and progressing through $250, $500, $1,000, etc., up to $1,000,000.
  6. Winning:
    • The user wins $1,000,000 by answering all 10 questions correctly.
  7. Incorrect Answer:
    • If the user answers a question incorrectly, they lose and win the amount corresponding to the last question they answered correctly.
  8. End of the Game:
    • The game ends when the user either answers a question incorrectly or wins the $1,000,000 prize.
